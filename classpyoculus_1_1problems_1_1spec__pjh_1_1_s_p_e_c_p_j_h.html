Detailed Description

Class that used to setup the SPEC Pressure Jump Hamiltonian system for interfacing Fortran.

For more detail, see the SPEC fortran module specpjh

Please note that in the Fortran module, \(\zeta\) is equivalent to \(\varphi\).

Pressure Jump Hamiltonian (PJH) Python module

Usage

pjh_problem = pyoculus.problems.SPECPJH(spec_output, lvol, dp, innout, plusminus)
Parameters
spec_datathe SPEC data generated by py_spec.SPECout
lvolwhich volume we are interested in, from 1 to spec_data.input.Mvol
dpthe \(\delta p\) for PJH
innout0 for inner surface, 1 for outer surface of the volume specified by lvol
plusminus– the sign for computing p_zeta, +1 or -1

For example, a SPEC equilibirum has 4 volumes. We are interested in computing PJH for the interface between the 2nd and the 3rd volume.

  • If we want to set the magnetic field on the inner side of this interface known, we need to choose lvol=2 and innout=1 (outer side of 2nd volume known)
  • On the other hand, if the field on the outer side of this interface is known, we choose lvol=3 and innout=0 (inner side of 3rd volume known)
  • The pressure jump \(\delta p\) should be choosen based on \(\delta p = 2 ( p_1 - p_2 ) = B_2^2 - B_1^2\), where \(B_1^2\) is the known field

Hint for choosing the initial p_theta and plusminus

Knowing the covariant components of the magnetic field on the known side of the interface could help. To do so, run 

Bco = get_covariant_field(theta, zeta)
Returns
a numpy array contains ( \( B_\theta, B_\zeta \)).

Physics

The pressure-jump Hamiltonian is derived from the force-balance condition at the ideal interfaces. Let \(p_1\) and \({\bf B}_1\) be the pressure and field immediately inside the interface, and \(p_1\) and \({\bf B}_1\) be the pressure and field immediately outside, then the force balance condition requires that

\[ H \equiv 2 \, \delta p = 2 ( p_1 - p_2 ) = B_2^2 - B_1^2 \]

For Beltrami fields, which satisfy \(\nabla \times {\bf B}=\mu {\bf B}\), the magnitude of the field, \(B\), on the interface (where we assume that \(B^s=0\)) may be written

\[ B^2 = \frac{g_{\zeta\zeta} f_\theta f_\theta - 2 g_{\theta\zeta}f_\theta f_\zeta + g_{\theta\theta} f_\zeta f_\zeta}{g_{\theta\theta}g_{\zeta\zeta}-g_{\theta\zeta}g_{\theta\zeta}} \]

where \(f\) is a surface potential and \(g_{\theta\theta}\), \(g_{\theta\zeta}\) and \(g_{\zeta\zeta}\) are metric elements local to the interface. \item Assuming that the field \(B_1\) is known on the ‘inside’ of the interface, ie. \(B_{1\theta}=f_\theta\), \(B_{1\zeta}=f_\zeta\) and \(f\) is known, it is required to determine the tangential field, \(p_\theta = B_\theta\) and \(p_\zeta = B_\zeta\), on the ‘outside’ of the interface. \item The o.d.e.'s are given by Hamilton's equations

\[ \dot \theta = \frac{\partial H}{\partial p_\theta}\Big|_{\theta,\zeta,p_\zeta}, \;\; \dot p_\theta = -\frac{\partial H}{\partial \theta}\Big|_{p_\theta,\zeta,p_\zeta}, \;\; \dot \zeta = \frac{\partial H}{\partial p_\zeta}\Big|_{\theta,p_\theta,\zeta}, \;\; \dot p_\zeta = -\frac{\partial H}{\partial \zeta}\Big|_{\theta,p_\theta,p_\zeta}, \]

where the ‘dot’ denotes derivative with respect to ‘time’.

This is reduced to a \(1\frac{1}{2}\) dimensional system by using \(\zeta\) as the time-like integration parameter, and replacing the equation for \(\dot p_\zeta\) with

\[ p_\zeta= P(\theta,p_\theta,\zeta; \delta p) = \frac{-b\pm\sqrt{b^2-4ac}}{2a} \]

where \(a=g_{\theta\theta}\), \(b=-2 g_{\theta\zeta}p_\theta\) and \(c=g_{\zeta\zeta} p_{\theta}^2 - (B_1^2 + 2 \, \delta p \,) G\) (see below for definition of \(G\)). The o.d.e.'s that then need to be followed are (see below for definition of \(A\) and \(b_2\))

\[ \frac{d \theta}{d\zeta}= \frac{g_{\zeta\zeta} p_{\theta} - g_{\theta\zeta} p_{\zeta}}{-g_{\theta\zeta}p_{\theta}+g_{\theta\theta}p_{\zeta}} \]

\[ \frac{d p_\theta}{d\zeta}= \frac{g_{\zeta\zeta,\theta} (-p_{\theta}^2)-2g_{\theta\zeta,\theta}(-p_{\theta}p_{\zeta}) +g_{\theta\theta,\theta}(-p_{\zeta}^2) + (B_1^2)_{\theta} G+ b_2 G_{\theta} / G } {-2g_{\theta\zeta}p_\theta+g_{\theta\theta}2p_{\zeta}}. \]

\[ G = g_{\theta\theta} g_{\zeta\zeta} - g_{\theta\zeta} g_{\theta\zeta} \]

\[ b_2 = g_{\zeta\zeta} p_{\theta}^2 - 2 g_{\theta\zeta} p_{\theta} p_{\zeta} + g_{\theta\theta} p_{\zeta}^2 \]

Note that \(d\theta / d \zeta = B^\theta / B^\zeta \); there is a fundamental relation between the pressure-jump Hamiltonian and the field-line Hamiltonian. (Furthermore, in many cases the surface will be given in straight field line coordinates, so \(d \theta / d\zeta = const.\).)
