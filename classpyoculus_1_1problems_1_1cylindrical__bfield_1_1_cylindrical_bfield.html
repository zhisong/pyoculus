Detailed Description

Class that used to setup the cylindrical bfield problem used in ODE solver.

The system of ODEs is given by

\[ \frac{dR}{d\phi} = \frac{B^{R}}{B^{\phi}} \]

\[ \frac{dZ}{d\phi} = \frac{B^{Z}}{B^{\phi}} \]
